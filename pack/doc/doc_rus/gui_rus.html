Описание

GUI.dll — это библиотека Lua, реализующая функции окон и виджетов, а также некоторые прочие функции.
Предназначена ТОЛЬКО для работы с редактором SciTE!
Примеры использования смотрим в файлах SideBar_tab*.lua и win_integrator.lua.
Вкладка Test, реализованная в файле SideBar_tab4.lua, активируется параметром 'sidebar.demo' со значением 1.

Подключение

Перед использованием функций библиотеки ее необходимо загрузить:

package.cpath = props["SciteDefaultHome"].."\\tools\\LuaLib\\?.dll;"..package.cpath
-- покажем все пути, по которым SciTE будет искать нашу библиотеку
print(package.cpath)
-- загрузка библиотеки
require 'gui'
-- тест успешной загрузки
if gui then
  for k,v in pairs(gui) do print(type(v), k) end
end

Функции

gui.version()

Описание:
  Функция, возвращающая строку с текущей версией библиотеки. (На момент написания документации - "0.2.8")

gui.colour_dlg()
gui.colour_dlg('#RRGGBB')

Описание:
  Запускает диалог выбора цвета. Возвращает числовое значение выбранного цвета или nil.
  Опциональный строковый параметр задает исходный цвет в формате '#RRGGBB'. По умолчанию '#000000' - чёрный цвет.

gui.pass_focus()

Описание:
  Переключаем фокус на редактор.

gui.get_ascii(128)

Описание:
  Функция для боковой панели. Получает число от 0 до 255.
  Возвращает соответствующий символ ASCII, html номер и код.   

local symbol, html_number, html_code = gui.get_ascii(i)

gui.day_of_year()
gui.day_of_year(day, month, year)

Описание:
  Без аргуменов возвращает номер текущего дня в году.
  С тремя целочисленными аргументами возвращает номер дня указанной даты.
  

print( gui.day_of_year(1, 3, 2024) ) --> 61
print( gui.day_of_year(1, 3, 2025) ) --> 60

gui.message('text', 'message')

Описание:
  Показывает окно с текстовым сообщением.
  Второй аргумент задаёт тип окна: "message", "warning", "error", "query".
  Возвращает логическое значение ответа.

gui.inifile('filename.ini', true)

Описание:
  Создаёт объект для работы с ini файлом.
  Первый аргумент задаёт путь к файлу или его имя.
  Второй аргумент задаёт - создать в текущей папке.

local ini = gui.inifile('filename.ini', true)
  Доступные методы:   
  ini:set_section('section name') -- устанавливает текущую секцию
  ini:get_section() - возвращает текущую секцию
  ini:write_string('key', 'string') -- записывает строку по указанному ключу
  ini:write_number('key', 123) -- записывает число по указанному ключу
  ini:read_string('key') -- читает строку по указанному ключу
  ini:read_number('key) -- читает число по указанному ключу
  ini:remove_section('section name') -- удаляет секцию
  ini:remove_key('key') -- удаляет ключ из текущей секции
  local str_path = ini:get_path() -- возвращает путь файла
  local tbl_keys = ini:get_keys() -- возвращает список ключей текущей секции
  local tbl_sections = ini:get_sections() -- возвращает список секций в файле.

gui.scite_window()

Описание:
  Возвращает объект окна редактора SciTE.
  Используется для прикрепления боковой панели.
  Также может быть использовано для получения/изменения размера и положения окна редактора и т.п.

print(gui.scite_window():position())
print(gui.scite_window():size())

gui.set_panel()

Описание:
  Только для редактора SciTE!
  Вызов без аргументов скрывает боковую панель, если она отображается.
  Иначе прикрепляет созданную панель со стороны, указанной вторым аргументом ("left" или "right")

local sb_panel = gui.panel(200)
gui.set_panel(sb_panel, "left" )

gui.panel()
gui.panel(width)

Описание:
  Только для редактора SciTE!
  Создаёт новую панель с указанной шириной. Ширина по умолчанию 100 пикселов.

gui.window()
gui.window("SideBar", style)

Описание:
  Только для редактора SciTE!
  Создаёт новое окно с указанным заголовком и стилем.
  Ширина по умолчанию 100 пикселов.
  Стиль по умолчанию WS_CAPTION + WS_SYSMENU. Подробнее о стилях.
  Иконка окна автоматически копируется из окна редактора.

Методы окон и панелей

wnd:show()

Описание:
  Показать окно или виджет.

wnd:hide()

Описание:
  Скрыть окно или виджет.

wnd:enable()
wnd:enable(bStatus)

Описание:
  Активирует или деактивирует виджет.

wnd:set_focus()

Описание:
  Устанавливает фокус на окно или виджет.

wnd:position()
wnd:position(posX, posY)

Описание:
  Получает или устанавливает позицию окна или виджета.

wnd:size()
wnd:size(w, h)

Описание:
  Получает или устанавливает размер окна или виджета.

wnd:center_h(child_window)

Описание:
  Центрирует виджет по ширине относительно другого виджета.

wnd:center_v(child_window)

Описание:
  Центрирует виджет по высоте относительно другого виджета.

wnd:center(child_window)

Описание:
  Выполняет wnd:center_v(child_window) и wnd:center_h(child_window)

wnd:resize(bStatus, nWidth, nHeight)

Описание:
  Включает\отключает изменение размеров окна. Устанавливает указанный размер.

wnd:bounds()

Описание:
  Возвращает 5 значений: отображается ли окно, координаты верхнего левого угла, ширина и высота окна.

wnd:client(widget)

Описание:
  Добавляется к окну wnd панель или окно widget с выравниванием "client".

wnd:add(widget, sAligment = "client", nSize = 100, bSplitted = true)

Описание:
  Добавляет к окну wnd панель или виджет widget с выравниванием sAligment ("top", "bottom", "left", "right" - пристаковано к указанной стороне,
  "none" - без стыковки, "client" - заполняет всё свободное место) и размером nSize (по умолчанию 100).
  Если задано выравнивание со стыковкой и установлен bSplitted, то будет создан сплиттер для изменения размеров дочернего окна.

wnd:remove_child(child_window)

Описание:
  Удаляет из окна wnd дочернее окно child_window.

wnd:context_menu{"..."}

Описание:
  Создаёт контекстное меню окна.

function list_test_menu1()
-- item1 handler
end
function list_test_menu2()
-- item2 handler
end
listbox:context_menu{"item1|list_test_menu1|item2|list_test_menu2|hideSideBar|".. (IDM_TOOLS + tonumber(props['CN_SIDEBAR']))}

wnd:statusbar(100, 200, -1)

Описание:
  Добавляет на окно панель состояния с блоками указанной ширины.

wnd:status_setpart(part_index, "text")

Описание:
  Устанавливает указанный текст в блок панели состояния с индексом part_index.

widget:get_ctrl_id()

Описание:
  Возвращает идентификатор виджета, заданный при его создании.

local Button_ID = 777
local btn = wnd:add_button(Button_ID, "simple tip")
btn:position(20, 420)
btn:size(70, 25)
wnd:set_tiptext(btn:get_ctrl_id(), "tiptext for button", true)

wnd:set_transparent( value )

Описание:
  Устанавливает прозрачность окна в диапазоне от 0 до 255.

wnd:remove_transparent()

Описание:
  Отключает прозрачность окна.

wnd:set_tiptext(ctrlID, "tiptext", "optCaption", optBoolBaloon, optBoolCloseBtn, optIconType)

Описание:
  Добавляет для виджета с идентификатором ctrlID всплывающее окно подсказки с указанным текстом.
  Опциональные аргументы: "optCaption" - заголовок (по умолчанию пустой),
  optBoolBaloon - стиль окна подсказки Baloon (по умолчанию false),
  optBoolCloseBtn - кнопка закрытия для стиля Baloon (по умолчанию false),
  optIconType - инонка окна подсказки для стиля Baloon (значения от 0 до 6, по умолчанию 0),

События окон и панелей

  Аргумент func методов для событий как правило является функцией или именем глобальной функции.
  Также допустимо указать число, соответствующее команде редактора SCI_*, или в виде строки команду меню "IDM_*".

wnd:on_close(func)

Описание:
  Устанавливает колбек на событие закрытия окна.

wnd:on_show(func)

Описание:
  Устанавливает колбек на событие включение отображения окна.

wnd:on_move(func)

Описание:
  Устанавливает колбек на событие перемещения окна.

wnd:on_focus(function(bIsFocused) end)

Описание:
  Устанавливает колбек на смену фокуса окна, получающий значение в фокусе окно или нет.

wnd:on_key(func)

Описание:
  Устанавливает колбек на нажатие кнопки. Колбек получает код нажатой клавиши, а также нажаты ли модификаторы Ctrl, Alt и Shift.

wnd:on_command(func)

Описание:
  Устанавливает колбек на нажатие контрола (кнопки, чекбокса). Колбек получает идентификатор нажатого контрола и его состояние.

wnd:on_scroll(func)

Описание:
  Устанавливает колбек на событие прокрутки. Колбек получает идентификатор виджета, в котором произошла прокрутка и позицию прокрутки.

wnd:on_size(func)

Описание:
  Устанавливает колбек на событие изменения размера окна. Колбек получает актуальную ширину и высоту окна.

wnd:on_timer(func, nOptPeriod)

Описание:
  Устанавливает колбек на срабатывание таймера с указанным периодом в секундах (по умолчанию 1 секунда)

wnd:stop_timer()

Описание:
  Отменяет работу ранее установленного таймера.

Описание:
  Устанавливает цвет текста и фона.

tree:set_tree_editable()

Описание:
  Разрешает редактировать названия узлов дерева.

tree:set_theme(bExplorer)

Описание:
  Устанавливает тему Explorer или Normal

tree:set_iconlib("sPath\\To\\icons.dll", bSmallSize)

Описание:
  Устанавливает набор иконок из указанной dll. Возвращает количество загруженных иконок. Аналогичный метод работает для tabbar и list.

tree:tree_get_item_selected()

Описание:
  Возвращает дескриптор выделенного узла.

tree:tree_set_insert_mode(hItem)
tree:tree_set_insert_mode(sMode)

Описание:
  Устанавливает режим добавления нового узла.
  Второй аргумент hItem - дескриптор узла или sMode - строка "last", "first", "sort", "root".

tree:tree_remove_item(hItem)

Описание:
  Удаляет указанный элемент.

tree:tree_get_item("sCaption", hItem)

Описание:
  Получить узел с указанным заголовком, прикрепленный в узлу hItem.

tree:tree_get_item_parent(hItem)

Описание:
  Получить родительский узел для указанного узла.

tree:tree_remove_childs(hItem)

Описание:
  Удаляет дочерние узлы для узла hItem.

tree:tree_get_item_text(hItem)

Описание:
  Получает название узла.

tree:tree_set_item_text(hItem)

Описание:
  Устанавливает название узла.

tree:tree_get_item_data(hItem)

Описание:
  Получить данные узла.

tree:tree_collapse(hItem)

Описание:
  Развернуть ветку.

tree:tree_expand(hItem)

Описание:
  Свернуть ветку.

tooltip:remove()

  Удаляет окно подсказки.

Методы рисования кистью и пером

tdc:color_back("#RRGGBB")

Описание:
  Задаёт цвет фона для текста. По умолчанию COLOR_BTNFACE.

tdc:color_text("#RRGGBB")
tdc:color_text(intR, intG, intB)

Описание:
  Задаёт цвет текста. По умолчанию "#000000".

tdc:text_align(nTextAlignFlags)

Описание:
  Задаёт выравнивание текста.

tdc:draw_text("text", pos_x, pos_y)

Описание:
  Отрисовать текст в указанной позиции.

tdc:set_pen("#RRGGBB", nWidth, nStyle)

Описание:
  Создаёт перо с указанными параметрами. Запоминает текущее перо.

tdc:reset_pen()

Описание:
  Возвращает запомненное перо.

tdc:xor_pen(true)

Описание:
  Включает стиль отрисовки.

tdc:solid_brush("#RRGGBB")

Описание:
  Создаёт сплошную кисть с указанным цветом. Запоминает текущую кисть.

tdc:hatch_brush("#RRGGBB", nStyle)

Описание:
  Создаёт штриховую кисть с указанным цветом и стилем nStyle (от 0 до 5).

tdc:set_pixel(posx, posy, "#RRGGBB")

Описание:
  Изменяет цвет пикселя с указанными координатами.

tdc:move_to(posx, posy)

Описание:
  Перемещает перо в указанные координаты.

tdc:line_to(posx, posy)

Описание:
  Проводит линию из текущей точки к точке с указанными координатами.

tdc:draw_line(x1, y1, x2, y2)

Описание:
  Рисует отрезок от точки (x1, y1) до точки (x2, y2).

tdc:polyline(x1, y1, x2, y2, ... , xn, yn)

Описание:
  Рисует ломаную с узлами в указанных точках (x1, y1), ..., (xn, yn).

tdc:polybezier(x1, y1, x2, y2, ... , xn, yn)

Описание:
  Рисует кривую Безье по указанным точкам (x1, y1), ..., (xn, yn).

tdc:rectangle(x1, y1, x2, y2)

Описание:
  Рисует прямоугольник с указанными вершинами.

tdc:polygone(x1, y1, x2, y2, ..., xn, yn)

Описание:
  Рисует замкнутый полигон с указанными вершинами.

tdc:ellipse(x1, y1, x2, y2)

Описание:
  Рисует эллипс, вписанный в прямоугольник с указанными вершинами.

tdc:round_rect(x1, y1, x2, y2, rw, rh)

Описание:
  Рисует прямоугольник с указанными вершинами со скруглением углов.
  Параметры rw, rh задают степень скругления в процентах от половины ширины и высоты соответственно.

tdc:chord(x1, y1, x2, y2, x3, y3, x4, y4)

Описание:
  Рисует сегмент эллипса, ограниченого хордой.
  Аргументы x1, y1, x2, y2 задают прямоугольник, в который вписан эллипс.
  Аргументы x3, y3 - начало ограничивающей хорды.
  Аргументы x4, y4 - конец ограничивающей хорды.

tdc:select_stock(nIndex)

Описание:
  Устанавливает системный объект отрисовки (перо, кисть, шрифт).
