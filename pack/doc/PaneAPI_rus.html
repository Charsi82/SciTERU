Scintilla API

Панели редактирования и вывода являются элементами управления Scintilla и могут управляться скриптами с использованием API Scintilla.

Функции API представлены здесь так, как они вызываются из Lua для панели редактирования. Все те же функции доступны для панели вывода, заменив «editor» на «output».

В Lua методы вызываются с помощью оператора ':', а доступ к свойствам осуществляется с помощью оператора '.' . Свойства доступны как для чтения, так и для записи по умолчанию, в противном случае они помечены как «только для чтения» или «только для записи».

Типы position и line представляют собой целочисленные типы, представляющие позиции и номера строк в документе. В 32-битных системах максимальное значение равно 2 147 483 647, но в 64-битных системах возможны и большие значения.

Тип «указатель» — это адрес памяти. Использовать этот тип из Lua сложно, но вполне возможно, что его можно передать обратно в Scintilla или в расширение, написанное на C.

string editor.TargetText Только для чтения.

position editor:ReplaceTarget(string text) -- Replace the target text with the argument text. Text is counted so it can contain NULs. Returns the length of the replacement text.

position editor:ReplaceTargetRE(string text) -- Replace the target text with the argument text after \d processing. Text is counted so it can contain NULs. Looks for \d where d is between 1 and 9 and replaces these with the strings matched in the last search operation which were surrounded by \( and \). Returns the length of the replacement text including any change caused by processing the \d patterns.

position editor:SearchPrev(int searchFlags, string text) -- Find some text starting at the search anchor and moving backwards. Does not ensure the selection is visible.

position editor.EdgeColumn -- Установить номер крайней колонки. Если текст выходит за край, он выделяется.

colour editor.EdgeColour -- Меняет цвет индикации границы.

editor:MultiEdgeAddLine(position column, colour edgeColour) -- Добавляет новую вертикальную границу в отображение строки.

editor:MultiEdgeClearAll() -- Очистить вертикальные границы.

position editor.MultiEdgeColumn[int which] Только для чтения.

Доступность

int editor.Accessibility -- Включить или отключить специальные возможности.

Лексер

int editor.Lexer Только для чтения. Индекс текущего лексера.

string editor.LexerLanguage Только для чтения. Имя текущего лексера

pointer editor.ILexer write-only -- Устанавливает лексер из ILexer*.

editor:Colourise(position start, position end) -- Раскрасить сегмент документа, используя текущий язык лексики.

int editor:ChangeLexerState(position start, position end) -- Указывает, что внутреннее состояние лексера изменилось в диапазоне и, следовательно, может потребоваться перерисовка.

string editor:PropertyNames() -- Возвращает список, свойств разделенных '\n' свойств, понимаемых текущим лексером. Результат оканчивается нулём.

int editor:PropertyType(string name) -- Возвращает тип свойства.

string editor:DescribeProperty(string name) -- Описание свойства. Результат оканчивается нулём.

string editor.Property[string key] -- Устанавливает свойство, использаемое лексером для какой-нибудь опциональной возможности.

string editor.PropertyExpanded[string key] Только для чтения.

int editor:GetPropertyInt(string key, int defaultValue) -- Возвращает "property" значение ранее заданное с помощью SetProperty, интерпретируемое как целое ПОСЛЕ замены значения "$()".

string editor.KeyWords[int keyWordSet] write-only -- Задает ключевые слова используемые лексером.

string editor:DescribeKeyWordSets() -- Возвращает список разделенных '\n' описаний наборов ключевых слов текущего лексера. Результат оканчивается нулём.

string editor.SubStyleBases Только для чтения.

int editor.DistanceToSecondaryStyles Только для чтения.

int editor:AllocateSubStyles(int styleBase, int numberStyles) -- Выделяет набор подстилей для модификации базового стиля, возвращает начало диапазона.

editor:FreeSubStyles() -- Освободить выделенные подстили.

int editor.SubStylesStart[int styleBase] Только для чтения.

int editor.SubStylesLength[int styleBase] Только для чтения.

int editor.StyleFromSubStyle[int subStyle] Только для чтения.

int editor.PrimaryStyleFromStyle[int style] Только для чтения.

string editor.Identifiers[int style] write-only -- Установить идентификаторы, которые отображаются в определенном стиле

pointer editor:PrivateLexerCall(int operation, pointer pointer) -- Для частной связи между приложением и известным лексером.

int editor.NamedStyles Только для чтения.

string editor:NameOfStyle(int style) -- Возвращает имя стиля. Результат оканчивается нулём.

string editor:TagsOfStyle(int style) -- Взвращает список тегов стиля, разделенных ' ' как "literal quoted string". Результат оканчивается нулём.

string editor:DescriptionOfStyle(int style) -- Возвращает описание стиля. Результат оканчивается нулём.

Уведомления

int editor.Identifier -- Установить идентификатор, сообщаемый как idFrom в сообщениях уведомления.

int editor.ModEventMask -- Установить, какие события модификации документа отправляются в контейнер.

bool editor.CommandEvents -- Укажите, будут ли события команд отправляться в контейнер.

int editor.MouseDwellTime -- Устанавливает время, в течение которого мышь должна оставаться неподвижной, чтобы сгенерировать событие задержки мыши.
